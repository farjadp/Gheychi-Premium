# Changelog

## [1.0.0] - V1 Premium Stable - 2026-04-16

نگارش ۱.۰.۰ کامل‌ترین و پایدارترین نسخه از سورس‌کد قیچی‌تولز است که آماده‌ی درآمدزایی (SaaS-Ready) می‌باشد. در این انتشار، علاوه بر سیستم دانلودر پایدار، تمام نیازهای ادمین برای مدیریت کسب‌و‌کار آنلاین توسعه داده شده است.

### 🌟 ویژگی‌های برجسته نسخه ۱ (SaaS Edition)
- **داشبورد گرافیکی (Visual GUI Builder):** امکان ویرایش پکیج‌ها، قیمت‌ها، نام‌ها و لیمیت‌ها بدون لمس کدهای خسته‌کننده JSON.
- **تم و گرافیک اپل (Glassmorphism):** رابط کاربری ادمین پنل با تم نیمه‌شفاف Midnight و فونت استاندارد Vazirmatn.
- **سیستم پرداخت آنلاین استرایپ (Stripe Integration):**
  - متصل به درگاه پرداخت هوشمند Stripe 
  - ایجاد خودکار فاکتور خرید (Checkout Session) برای کاربران از محیط تلگرام
  - وب‌هوک (Webhook) امن و سریع جهت اِعمال اتوماتیک اشتراک پس از پرداخت دلاری
- **مدیریت API از راه دور (Advanced Settings):** امکان روشن/خاموش کردن اتصال به Cobalt API و RapidAPI مستقیماً از تنظیمات بدون نیاز به دیپلوی.
- **سیستم پشتیبان‌گیری سرد (Hardware Backup):** گرفتن فول‌بک‌آپ درجا (.zip) از کل کدهای سیستم و دیتابیس کاربران فقط با فشردن یک دکمه.

### 🚀 دانلودر هسته
- **معماری سه‌لایه (3-Layer Fallback Architecture)** برای مقاومت ۱۰۰٪ در برابر مسدودسازی شبکه‌های اجتماعی:
  - **لایه اول:** دسترسی هوشمند با `Cobalt API` (سریع‌ترین مسیر).
  - **لایه دوم:** هدایت از طریق دو مسیرِ تفکیک‌شده در `RapidAPI` (یکی مخصوص YouTube و یکی برای Auto Download All In One مخصوص مدیاهای دیگر).
  - **لایه سوم:** مکانیزم Fallback ایمن با `yt-dlp` به عنوان راهکار نهایی.
- دور زدن هوشمند دیوار آتش توییتر و اینستاگرام و یوتیوب.
- پشتیبانی رسمی از ده‌ها پلتفرم معتبر شامل YouTube, Instagram, X/Twitter, PornHub, RadioJavan, SoundCloud, TikTok.

### 🔒 امنیت و ساختار
- جداسازی کامل تنظیمات از فایل‌های اصلی. تمامی تنظیمات درون دیتابیس `settings.json` با ساختار خوانا ذخیره می‌شوند.
- جلوگیری از اجرای دوگانه‌ی کانتینرها (رفع خطای `409 Conflict`).
