# Changelog

## [1.1.0] - Payment Notifications & User List Improvements - 2026-05-10

### 🎉 اعلان پرداخت دو زبانه (Bilingual Payment Notifications)
- **پیام تأیید خودکار پس از پرداخت Stripe:** به محض موفقیت پرداخت، کاربر پیام کامل فارسی و انگلیسی دریافت می‌کند شامل:
  - نام پلن فعال شده
  - تاریخ انقضا (YYYY-MM-DD UTC)
  - مبلغ پرداختی
  - لیست کامل قابلیت‌های پلن (سهمیه هر پلتفرم)
  - لینک‌های دسترسی سریع (/menu, /usage, /myplan, /dashboard)
- **تأیید دستی تراکنش:** در پنل ادمین، وقتی تراکنش معلق را تایید می‌کنی، همین پیام دو زبانه برای کاربر ارسال می‌شود.
- **Helper مشترک:** `_activate_paid_plan()` در `admin_panel.py` که از هر دو مسیر (webhook و manual) استفاده می‌شود.

### 👥 بهبود لیست کاربران در پنل ادمین (Admin Panel User List)
- **Pagination سمت سرور:** نمایش 20 کاربر در هر صفحه با دکمه‌های قبلی/بعدی
- **تاریخ عضویت:** ستون جدید که نشان می‌دهد کاربر کی ثبت‌نام کرده (`created_at`)
- **آخرین استفاده:** ستون جدید که نشان می‌دهد آخرین باری که کاربر دانلود انجام داده کی بوده (`last_usage`)
- **شمارش کل کاربران:** تعداد کل در عنوان کارت نمایش داده می‌شود
- **API پشتیبانی:** `/api/users` الان با پارامترهای `page` و `per_page` کار می‌کند

### 🔧 تغییرات داخلی
- **`runtime_store.py`:**
  - اضافه شدن `get_user_last_usage()` برای محاسبه آخرین timestamp استفاده کاربر
  - اضافه شدن `count_bot_users()` برای شمارش کل کاربران
  - به‌روزرسانی `list_bot_users()` برای پشتیبانی از `offset` و شامل کردن `last_usage`
- **`admin_panel.py`:**
  - Helper جدید `_format_payment_success_message()` برای ساخت پیام دو زبانه
  - Helper جدید `_send_telegram_message()` برای ارسال async از داخل Flask
  - Helper جدید `_activate_paid_plan()` برای فعال‌سازی یکپارچه پلن + ارسال پیام
  - به‌روزرسانی `admin_index()` برای پشتیبانی از pagination
  - به‌روزرسانی `/api/users` برای پشتیبانی از pagination
  - به‌روزرسانی `/finance/confirm` برای ارسال پیام تأیید
  - ساده‌سازی `stripe_webhook` برای استفاده از helper مشترک
- **`locales.py`:**
  - کلیدهای جدید: `payment_success_title`, `payment_success_body`, `payment_success_unlimited_expiry` در fa و en

## [1.0.0] - V1 Premium Stable - 2026-04-16

نگارش ۱.۰.۰ کامل‌ترین و پایدارترین نسخه از سورس‌کد قیچی‌تولز است که آماده‌ی درآمدزایی (SaaS-Ready) می‌باشد. در این انتشار، علاوه بر سیستم دانلودر پایدار، تمام نیازهای ادمین برای مدیریت کسب‌و‌کار آنلاین توسعه داده شده است.

### 🌟 ویژگی‌های برجسته نسخه ۱ (SaaS Edition)
- **داشبورد گرافیکی (Visual GUI Builder):** امکان ویرایش پکیج‌ها، قیمت‌ها، نام‌ها و لیمیت‌ها بدون لمس کدهای خسته‌کننده JSON.
- **تم و گرافیک اپل (Glassmorphism):** رابط کاربری ادمین پنل با تم نیمه‌شفاف Midnight و فونت استاندارد Vazirmatn.
- **سیستم پرداخت آنلاین استرایپ (Stripe Integration):**
  - متصل به درگاه پرداخت هوشمند Stripe 
  - ایجاد خودکار فاکتور خرید (Checkout Session) برای کاربران از محیط تلگرام
  - وب‌هوک (Webhook) امن و سریع جهت اِعمال اتوماتیک اشتراک پس از پرداخت دلاری
- **مدیریت API از راه دور (Advanced Settings):** امکان روشن/خاموش کردن اتصال به Cobalt API و RapidAPI مستقیماً از تنظیمات بدون نیاز به دیپلوی.
- **سیستم پشتیبان‌گیری سرد (Hardware Backup):** گرفتن فول‌بک‌آپ درجا (.zip) از کل کدهای سیستم و دیتابیس کاربران فقط با فشردن یک دکمه.

### 🚀 دانلودر هسته
- **معماری سه‌لایه (3-Layer Fallback Architecture)** برای مقاومت ۱۰۰٪ در برابر مسدودسازی شبکه‌های اجتماعی:
  - **لایه اول:** دسترسی هوشمند با `Cobalt API` (سریع‌ترین مسیر).
  - **لایه دوم:** هدایت از طریق دو مسیرِ تفکیک‌شده در `RapidAPI` (یکی مخصوص YouTube و یکی برای Auto Download All In One مخصوص مدیاهای دیگر).
  - **لایه سوم:** مکانیزم Fallback ایمن با `yt-dlp` به عنوان راهکار نهایی.
- دور زدن هوشمند دیوار آتش توییتر و اینستاگرام و یوتیوب.
- پشتیبانی رسمی از ده‌ها پلتفرم معتبر شامل YouTube, Instagram, X/Twitter, PornHub, RadioJavan, SoundCloud, TikTok.

### 🔒 امنیت و ساختار
- جداسازی کامل تنظیمات از فایل‌های اصلی. تمامی تنظیمات درون دیتابیس `settings.json` با ساختار خوانا ذخیره می‌شوند.
- جلوگیری از اجرای دوگانه‌ی کانتینرها (رفع خطای `409 Conflict`).
